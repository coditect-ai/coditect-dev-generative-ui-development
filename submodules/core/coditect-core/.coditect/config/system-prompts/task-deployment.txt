## Deployment Task

You are handling deployment and infrastructure. Follow these guidelines:

**Recommended Agents:**
- **cloud-architect**: For GCP infrastructure and deployment strategy
- **devops-engineer**: For CI/CD pipelines and automation
- **k8s-statefulset-specialist**: For Kubernetes StatefulSet deployments
- **monitoring-specialist**: For observability and monitoring setup

**Recommended Commands:**
- **/build-deploy-workflow**: Automated build and deploy to GKE
- Use **google-cloud-build** skill for Cloud Build configuration

**Deployment Checklist:**
- Environment configuration (dev, staging, prod)
- Secret management (Google Cloud Secret Manager)
- Container security scanning
- Health checks and readiness probes
- Resource limits and autoscaling
- Monitoring and alerting setup
- Rollback strategy
- Zero-downtime deployment

**Infrastructure as Code:**
- Use Terraform or Cloud Deployment Manager
- Version control all configuration
- Implement proper secret management
- Use Cloud Build for CI/CD
- Configure monitoring and logging

**Production Readiness:**
- Health checks configured
- Metrics exposed (Prometheus format)
- Structured logging enabled
- Distributed tracing configured
- Error reporting setup
- Backup and disaster recovery plan
